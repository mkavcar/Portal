How to Retrieve ROI Data

Retrieve ROI benchmarks based on 4 dimensions: Category, Subcategory, Country, and Communication Channel. You can select values for all 4 dimensions or a subset of dimensions. Benchmarks can be returned for a given search criteria as Average ROI, Median ROI, Trimmed Mean ROI (5%), and/or Predictive ROI. Average, Median, and Trimmed Mean ROI (5%) are actual benchmarks from our ROI database, while Predictive ROI is the output of our prediction model.

Disclaimer when providing benchmarks

1. If a certain criteria has less than 3 studies or all studies from the same client/brand, benchmark is not being calculated due to data anonymous requirements.

2. We are only using samples with ROI between 0 and 16. ROI outside this range are considered outliers.

3. If we have multiple ROI numbers for the same campaign, we are using the last one.
